今日头条面试经验

发表于 2018-03-08

面试前

转眼就开学啦，寒假结束前的晚上在家刷北邮人论坛，看到一个这样的帖子《现在都没有做安卓的同学了吗？》，作为一个做安卓的同学自然点进去了。发帖人是实习版的版主，现在在头条的学长，据他说，现在安卓很缺人，招不到人，还有20+个名额！

看得我很是兴奋哈，赶紧回帖问了要不要研一的实习生，对方强烈表示要的！兴奋之余去牛客网看了一下头条的笔试题，很快就冷静下来了。。都不会！！！

不得不说，现在安卓开发比想象的要抢手，很多人跟风去学数据挖掘之类的了。因为我在帖子里回复之后，又有学长主动私信我要不要去他那里实习（他在keep），问了一下，keep的要求好像低一点，当天晚上投了简历过去。不过，当时以为这学期还有一些课，一周很难保证4天实习，就如实说了。也不知道是不是这个原因，直到今天我还没收到面试邀请（😣）也许是简历挂了吧

后来来到学校，选课的时候发现可以把课都放在周一，这样其他天就可以出去实习啦！然后找实习的欲望就越烧越旺。。。
其实，最想去的是小米（毕竟算是个米粉😄），在论坛上找了几个小米的android实习，投了简历，不过也没回复~

然后又下载了“实习僧”app，在app上随便投了好几家（京东，搜狐，小米，，，）不过都没收到面试邀请😣

好吧！原来一个面试机会都这么难！！找实习确实不容易啊！！！

实习僧还是靠不住！果断去论坛找，发现有学长在那篇帖子下吐槽同样招不到人。果断私信他，对方表示要安卓实习生，他是网易有道的，给我内推了，不过目前还没收到面试。。可能简历又挂了(┬＿┬))

就在我一边看着安卓基础知识，一边焦急的翻着邮箱的时候，头条hr给我打电话了！让我3月7号去面试！当时其实还是很开心的(^__^) 嘻嘻)。然后去网上搜了一下头条的一些面经，，又一次凉了。面试难度普遍反应较大，而我只是看了两三天的安卓知识，算法也没刷。。。

在室友和朋友的怂恿下（其实自己也不想放弃这个机会，哪怕面不上，涨涨经验也很好），硬着头皮就决定去了

写写日记，记录一下学习状态

今天天气不好，这两天都是，阴雨天，有点凉。

开学以来快两个月了，在北邮的生活还算愉快，虽然和想象中的研究生生活有点出入（大概是被本科老师误导的）
很开心，上个星期四的面试通过了，这样我拿到了研究生生涯的第一份实习工作。挺感激我们的导师的，虽然因为扩招的原因，导致老师一个人要管理很多学生。但是杨老师还是很复杂的周一开会让我们汇报学习情况，对比同寝室的其他导师，确实要好一些。而且平时实验室不要求考勤，居然还主动内推实习！不然我的实习估计还通过不了呢
好吧，都快变成我们实验室安利文了。
实验室根据个人兴趣进行了分组，热门的spark大数据组没抢上，来了感兴趣的Android组。说来我一直对Android有兴趣，也许以后就弄Android了。
对于Android开发，以前都是自己兴趣开发，没有系统的学习过。基本上是面向百度、谷歌编程。现在学姐每周会给我们布置一些任务，完成这些任务，并且制作ppt，周一上午汇报给导师听，感觉确实学到了不少东西。
说说上周四的面试吧，因为是导师内推，实验室和公司合作的，所以面试简单了很多。5个同学一起去的，没有一个一个的面试，而是一起面试的。因为我简历上没有大数据开发的经历，只能到前端去了（这次只有这两个职位）。可能对于前端的需求会比较少吧，我被问到的问题也不多，主要问题是围绕着大数据来的。
话说面试前，我准备了两天，有准备基础算法知识和前端知识，居然一个也没问道，也是比较惨。。面试这种还真的是很难零时抱佛脚。
明天下午要去弄入职手续，也许这是为数不多的悠闲的周末了。感觉以后会很忙，要忙着实习工作，实验室任务，课堂任务，还有算法学习任务。。。
不过，我感觉忙一点才充实嘛，闲着的时候我也只是在浪费时间

(leetcode)496. Next Greater Element I

发表于 2017-10-09

(leetcode)496. Next Greater Element I

题目

You are given two arrays (without duplicates) nums1 and nums2 where nums1’s elements are subset of nums2. Find all the next greater numbers for nums1's elements in the corresponding places of nums2.

The Next Greater Number of a number x in nums1 is the first greater number to its right in nums2. If it does not exist, output -1 for this number.

Example 1:
Input: nums1 = [4,1,2], nums2 = [1,3,4,2].
Output: [-1,3,-1]
Explanation:
    For number 4 in the first array, you cannot find the next greater number for it in the second array, so output -1.
    For number 1 in the first array, the next greater number for it in the second array is 3.
    For number 2 in the first array, there is no next greater number for it in the second array, so output -1.
Example 2:
Input: nums1 = [2,4], nums2 = [1,2,3,4].
Output: [3,-1]
Explanation:
    For number 2 in the first array, the next greater number for it in the second array is 3.
    For number 4 in the first array, there is no next greater number for it in the second array, so output -1.
Note:
All elements in nums1 and nums2 are unique.
The length of both nums1 and nums2 would not exceed 1000.

这题下午在看的时候感觉能做，晚上回来就被打脸了。。。我能想到的解法自然是比较基础的遍历两个数组，从第一个数组中拿一个数到第二个数组中做比较，先是找到这个数字，然后看这个数字后面有没有比它更大的数字，如果有，就放到结果集里，如果没有，就放-1到结果集中。
